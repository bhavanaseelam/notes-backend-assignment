# Notes Backend Assignment

A backend Notes application built using FastAPI, SQLite, and JWT Authentication.

## Features

- User Registration
- User Login with JWT Authentication
- Create Notes
- Get All Notes
- Get Single Note
- Update Notes
- Delete Notes
- Share Notes with Other Users
- OpenAPI Documentation
- About Endpoint

## Tech Stack

- FastAPI
- SQLite
- SQLAlchemy
- JWT Authentication
- Render Deployment
